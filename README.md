
## 🛠 Tecnologías Utilizadas
- **Lenguaje**: Java
- **IDE**: :inlineEntity{type="inline_entity" conversation="090995ea37a6a0debc060eb12fbf6597e2b8" name="Apache&#32;NetBeans&#32;25"}
- **Gestión de dependencias**: Maven (`pom.xml`)
- **Control de versiones**: Git / GitHub

## 📝 Uso
Este proyecto permite:
- Registrar vehículos mediante herencia (clases hijas de `Vehículo`).
- Almacenarlos en una lista dinámica (`ArrayList`).
- Gestionar datos a través de formularios gráficos.

## 🌟 Características
- ✅ Uso de encapsulamiento y herencia.
- ✅ Manejo de colecciones en Java.
- ✅ Diseño modular con paquetes.
- ✅ Interfaz gráfica básica (opcional).
